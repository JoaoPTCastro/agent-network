# 🐙 Agent Network

> Dashboard visual para gerenciar sua rede de agentes de IA — inspirado no [Paperclip](https://paperclip.ing/).

## Visão Geral

Agent Network é uma interface web que simula uma "empresa virtual" onde você é o CEO e seus agentes de IA trabalham de forma coordenada para alcançar seus objetivos.

## 🎯 Missão

**Sair da CLT, alcançar R$15K+/mês, aposentadoria aos 35**

## 🏢 Organograma

```
João (CEO)
    │
    └── B.K. (COO) — Coordenação geral
            │
            ├── Dev — Código e automações
            ├── Marketer — Marketing e conteúdo
            ├── Researcher — Pesquisa e análise
            └── Ops — Operações rotineiras
```

## 🚀 Projetos

1. **Loja de Atacado (Rifa)** — CLT atual, laboratório pra automação
2. **Agência de Marketing** — Serviços de marketing e tráfego
3. **SaaS com IA** — Produto replicável com renda passiva

## 📋 Funcionalidades

- 📊 **Organograma visual** — Veja sua equipe de agentes
- 🚀 **Projetos** — Acompanhe progresso dos seus projetos
- 📋 **Tarefas** — Gestão de tarefas comchecklist
- ⚡ **Ações rápidas** — Deploy, GitHub, novos agentes
- 💾 **Persistência local** — Salva estado no navegador

- HTML/CSS/JS puro — sem dependências
- GitHub Pages — hospedagem gratuita
- LocalStorage — persistência de estado

## 📦 Deploy

```bash
# Clone o repo
git clone https://github.com/JoaoPTCastro/agent-network.git
cd agent-network

# Faça suas alterações...
# Push para GitHub
git push origin main
# Ative GitHub Pages em Settings > Pages > Source: main
```
