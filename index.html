FAQ

How is Agent Network different from agents like OpenClaw?
Agent Network uses those agents. It orchestrates them into a team — with org charts, budgets, goals, governance, and accountability.
Can I use my existing agents?
Yes. Agent Network is unopinionated about agent runtimes. Your agents can be OpenClaw bots, Claude sessions, Python scripts, shell commands, HTTP webhooks — anything that can receive a heartbeat signal.
What happens when an agent hits its budget limit?
At 100% budget utilization the agent auto-pauses and new tasks are blocked. You get a soft warning at 80%. As the board, you can override the limit at any time and resume the agent.
How do I keep agents from doing things I don't want?
Agent Network has governance and control modules. Hiring new agents is gated by Board approval by default. You have full control over every agent in the org.
Why should I use this instead of just using OpenClaw?
Agent orchestration has subtleties in how you coordinate who has work checked out, how to maintain sessions, monitoring costs, establishing governance — Agent Network does this for you.
